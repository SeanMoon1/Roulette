## 💡 주요 기능 설명

### ✨ 기능 1: \[룰렛 기능 구현]

* 초기값 상태에선 빈 룰렛이 나오고, 사용자 입력에 따라 항목이 룰렛에 출현하도록 구현

### ✨ 기능 2: \[룰렛 이벤트 기능]

* 룰렛 항목 수량 조절 버튼
* Enter키로 항목 추가 가능

### ✨ 기능 3: \[룰렛 구동 기능]

* Start 버튼과 Stop 버튼으로 룰렛의 시작과 끝 동작 가능능
* 당첨된 항목 팝업으로 띄우기기
